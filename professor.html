Carregar submissões

O painel abre com dados de demonstração. Escolha uma das duas formas de carregar dados reais.

JSON

Opção A — Carregar do Apps Script

Requer o token docente (configurado uma vez via setDocenteToken() em Setup.gs — nunca fica no código-fonte). O token só fica salvo neste navegador enquanto a aba estiver aberta.

Opção B — Colar JSON manualmente

Alternativa sem token: rode exportSubmissoesJSON() no editor do Apps Script e cole o resultado do log abaixo.

Identificação não pontua
O status de identidade (nome cotejado com a turma) não soma nem subtrai da nota técnica — identificar-se corretamente é uma condição básica da entrega, não um mérito avaliável.
